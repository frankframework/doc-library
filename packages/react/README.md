# Doc Library React Implementation

### Serve
Run `pnpm run dev` to build the library project first and then serve the playground.  
Your application will be available at `http://localhost:5173`.

### Build
Run `pnpm build` to build the library project using Vite. The build artifacts will be stored in the `dist/` directory.

### Publishing
Run `pnpm publish` in the `dist/` directory in order to publish the library to a package registry.
