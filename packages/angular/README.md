# Doc Library Angular Implementation

### Serve
Run `pnpm start` to build the library project first and then serve the playground.  
Your application will be available at `http://localhost:4200`.

### Build
Run `pnpm build` to build the library project using ng-packagr. The build artifacts will be stored in the `dist/ff-doc/` directory.

### Publishing
Run `pnpm publish` in the `dist/ff-doc/` directory in order to publish the library to a package registry.
